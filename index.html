node-bus by node-bus

A pubsub event bus in node that facilitates events between browser clients and the server.

Dependencies

Node.js

Install

Unzip/tar the source archive, and use the files stored in ./src (bus.client.js and bus.server.js) in your Node.js project. Within your project, require('bus.server') and pipe one of your URL endpoints to node-bus by sending the HTTP request and response objects for a request to bus.service(req, res);
